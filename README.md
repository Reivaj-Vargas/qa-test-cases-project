# QA Test Cases Project – Reivaj Vargas

## 📌 Descripción del proyecto
Este proyecto consiste en el diseño y ejecución de pruebas manuales para una aplicación web, incluyendo la creación de casos de prueba, validación de funcionalidades y detección de errores.

## 🎯 Objetivo
Asegurar la calidad de la aplicación mediante pruebas funcionales, identificando fallos y validando el correcto funcionamiento del sistema.

## 🧪 Actividades realizadas
- Diseño de casos de prueba
- Ejecución de pruebas funcionales
- Validación de interfaz de usuario (UI)
- Pruebas en distintos navegadores (cross-browser)
- Identificación y documentación de bugs

## 🛠️ Herramientas utilizadas
- Jira
- Postman
- DevTools
- Excel

## 📂 Contenido del repositorio
- `test-cases/` → Casos de prueba diseñados
- `bug-reports/` → Reporte de errores detectados
- `api-testing/` → Pruebas de API (Postman)

## 🚀 Resultados
- Identificación de errores funcionales en la aplicación
- Validación de flujos principales del usuario
- Mejora en la calidad del sistema a través del testing

## 👨‍💻 Sobre mí
QA Engineer Junior en formación, enfocado en testing de aplicaciones web y APIs, con experiencia en diseño de pruebas y documentación de errores.
